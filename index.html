Proyecto ID Cultural

Plataforma digital dedicada a visibilizar, preservar y promover la identidad artística y cultural de Santiago del Estero.

El Problema: El valioso legado artístico de Santiago del Estero estaba fragmentado, corriendo el riesgo de perderse por la falta de una plataforma digital centralizada.

Mi Rol: Como arquitecto de la solución y líder del equipo técnico de RunaTech Dev, mi rol fue guiar la estrategia de desarrollo y asegurar la construcción de una base de código robusta y escalable para el proyecto.

Tecnologías Utilizadas:

PHP MariaDB JavaScript HTML5 CSS3 Bootstrap

El Impacto: La plataforma se encuentra en fase de desarrollo avanzada, con el objetivo de convertirse en el archivo digital definitivo para la cultura local, asegurando que el legado de nuestros artistas sea inmortal.

REGroup - E-Waste Sustentable ("Tachy")

Un proyecto de triple impacto que busca revolucionar la gestión de e-waste. 'Tachy' es un ecosistema integral que combina una aplicación móvil y un programa de incentivos para fomentar el reciclaje de forma social, ambiental y económica.

El Problema: La gestión de e-waste en la región era ineficiente y carecía de participación ciudadana. No existía un sistema integrado que incentivara el reciclaje de forma efectiva.

Mi Rol: Como Fundador y Estratega de REGroup, comando a un equipo multidisciplinario en la misión de definir y construir el sistema "Tachy". Mi función es traducir la visión en un plan de batalla técnico y operativo.

Tecnologías Utilizadas:

Python Flask SQL Bootstrap HTML5 CSS3

El Impacto: El prototipo funcional fue presentado en el Rally de Innovación Santiago del Estero 2025. Actualmente, el equipo se encuentra en la fase de desarrollo del primer producto viable, con el objetivo de revolucionar el paradigma del reciclaje.

Altoke Servicio Técnico – Página Web

Diseño y desarrollo de una plataforma web moderna para la transformación digital de un servicio técnico, con el objetivo de expandir su alcance y evolucionar hacia una futura tienda online.

El Problema: Un exitoso servicio técnico local carecía de una presencia digital profesional, limitando su alcance y capacidad para competir en el mercado online.

Mi Rol: Como co-fundador de este emprendimiento junto a mi socio en Jujuy, actúo como Líder de Proyecto y Consultor Estratégico. Mi misión es guiar la transformación digital de nuestro propio servicio técnico.

Tecnologías Propuestas:

PHP SQL JavaScript Bootstrap HTML5 CSS3

El Impacto: El resultado será una nueva y poderosa embajada digital para nuestra marca, posicionándola para capturar un nuevo segmento de mercado y asegurar su crecimiento en la era digital.
